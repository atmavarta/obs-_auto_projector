# Auto Program Projector for OBS

Script ini dibuat untuk **secara otomatis membuka Program Projector** pada OBS Studio ke monitor tertentu.  
Cocok digunakan untuk gereja, studio kecil, kelas, atau event yang membutuhkan tampilan **layar Program** otomatis saat OBS dibuka atau saat scene tertentu aktif.

## ✨ Fitur

- **Auto membuka Program Projector** saat OBS selesai loading.
- **Bisa membuka projector hanya saat masuk scene tertentu** (opsional).
- **Memaksa output ke Program (bukan Preview)** sehingga tidak salah projector.
- **Mendukung OBS 28 – 31**.
- **Monitor bisa dipilih manual melalui pengaturan script**.
- Script ringan dan berjalan otomatis di background.
